# TLDW
Answer the call and let the journey ignite. 

The path to the Last Dragon opens only to those who possess them all.

A 10-20 minute adventure for vanilla Minecraft 1.21.5.

Tested on 1-3 players.

**Multiplayer**
In order to work, the server.properties must have the following settings enabled:

- enable-command-block=true

Other recommended settings:
- difficulty=hard
- gamemode=adventure
- simulation-distance=12
- view-distance=12
