# TLDW
Answer the call and let the journey ignite. 

The path to the Last Dragon opens only to those who possess them all.

Find spells and challenge the tower. 

---

A 10-20 minute adventure for vanilla Minecraft 1.21.6.

Anyplayer map, tested on 1-3 players.

Recommended render distance of 12.

[Ctarron discord](https://discord.gg/jbTWxc4jjv) server for feedback/doubts.

### Multiplayer
Extract the resource pack from the world folder and share it with the other players.
In order to work, the server.properties must have the following settings enabled:
- enable-command-block=true

Other recommended settings:
- difficulty=hard
- gamemode=adventure
- simulation-distance=12
- view-distance=12

Available translations:
- English
- Spanish

Para jugar en español, cambia el idioma del juego a Español.
